How to Process a Return in Return It

This guide provides step-by-step instructions on how to efficiently process a return using the Return It platform. Follow these steps to ensure a smooth and accurate return process.

Step-by-Step Guide

  • Locate the Return Request: Navigate to the 'Returns' section in your Return It dashboard. Use the search bar to find the specific return request by order number, customer name, or other relevant details.
  • Review Return Details: Click on the return request to view detailed information, including the reason for return, items being returned, and customer notes. Verify that all information is accurate and complete.
  • Authorize the Return: If the return meets your return policy criteria, click the 'Authorize Return' button. You may need to provide a return shipping label or instructions for the customer.
  • Receive the Returned Items: Once the customer ships the items back, track the shipment using the provided tracking number. Upon receiving the items, inspect them to ensure they are in the condition described in the return request.
  • Process the Refund or Exchange: If the items are in acceptable condition, proceed with processing the refund or exchange. Use the Return It platform to initiate the refund or create a new order for the exchange.
  • Complete the Return: After processing the refund or exchange, mark the return as 'Completed' in the Return It system. This will update the return status and notify the customer of the completion.
